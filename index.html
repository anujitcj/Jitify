Get Your Free API Key 100% Free

Groq gives you a generous free tier — no credit card needed. Takes about 2 minutes.

1

Go to Groq Console

Visit console.groq.com and click Sign Up. You can sign up with Google or GitHub — no credit card required.

2

Create an API Key

After logging in, go to API Keys in the left sidebar. Click Create API Key, give it any name (e.g. "Jitify"), then copy the key — it starts with gsk_.

3

Paste it into Jitify

Paste your key into the Groq API Key bar at the top of the page. The status dot will turn green and you're ready to humanize.

4

Free Tier Limits

Groq's free tier includes ~14,400 requests/day on Llama 3 models — more than enough for heavy daily use. No expiry.
